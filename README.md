LivelyPy
====================
LivelyPy is a python text editor prototype to an auto-compile editor.

![Alt text](sample.png)

## Requirements:
+ gtksourceview2

Basic Usage
---------------------
## Running:
> python main.py

## Commands

### File Operations
+ New => Ctrl + N
+ Close => Ctrl + W
+ Save => Ctrl + S
+ Open => Ctrl + O

### Autocomplete
+ Select => Alt + Enter
+ Move Up => Alt + Up
+ Move Down => Alt + Down

### Evaluate
Evaluate selected text or cursor location
+ Ctrl + E
